# Sudoku-Solver
Puzzle solver
Sudoku is  puzzle in which players insert the numbers one to nine into a grid consisting of nine squares subdivided into a further nine smaller squares in such a way that every number appears once in each horizontal line, vertical line, and square.
The program is coded in Java. This program uses key back tracking concepts. 
During the making of this program back-tracking has been used fully to understand it working and applying it.
